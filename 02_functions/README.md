# Functions

In this module, we will learn about how to become even lazier programmers through the use of creating our own functions. We will learn how to define our own functions, return data, variable scope, and lifetime within a function and set default variables in the function arguments.  Additionally, we will discuss how to document our code. 

**No, you don't have to read or watch everything. Pick the medium(s) that fit
your learning style.**

## Key Concepts

- Functional Parameters
- Function documentation
- Function return values
- Default parameters
